# BedrockELA Mobile App

This is the **mobile app version** of BedrockELA, built to publish to iOS App Store and Google Play Store.

## 📱 Project Structure

```
bedrockela-app/
├── www/                    # Web assets (HTML, CSS, JS)
├── ios/                    # iOS native project (generated by Capacitor)
├── android/                # Android native project (generated by Capacitor)
├── manifest.json           # PWA manifest
├── sw.js                   # Service worker for offline support
├── capacitor.config.json   # Capacitor configuration
└── package.json            # Dependencies
```

## 🛠 Commands

```bash
# Install dependencies (when ready)
npm install

# Run PWA locally
npm run serve

# Build for production
npm run build

# Add iOS platform
npx cap add ios

# Add Android platform
npx cap add android

# Sync web → native
npx cap sync

# Open in Xcode (iOS)
npx cap open ios

# Open in Android Studio
npx cap open android
```
